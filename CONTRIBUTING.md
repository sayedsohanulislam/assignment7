Thanks for checking out KeenKeeper! If you'd like to contribute, please follow these simple steps:

1. Fork the repository on GitHub.
2. Create a feature branch: `git checkout -b feature/my-change`.
3. Make changes with clear commit messages.
4. Run `npm install` and `npm run dev` to verify the app runs.
5. Open a Pull Request describing your changes.

Be respectful in code reviews and keep changes focused and small.

Thank you!
